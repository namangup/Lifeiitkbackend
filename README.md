# Acads API

### All Acads Details
View all Acads information
```
Endpoint: /api/acads/all/
Method: GET
Authorization Not Required
Response:
[
    {
        "course_id": 1,
        "name": "CS",
        "code": "CS101"
    }
]
Status Code: 200_OK
```
